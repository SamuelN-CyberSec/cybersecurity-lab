## 🛠️ Platforms to Target

### Direct Hire / Full-Time Remote
1. **LinkedIn** — Optimize profile, use "Open to Work" (remote)
2. **WeWorkRemotely** — Filter: Security/DevOps
3. **RemoteOK** — Keyword: Security
4. **Dice.com** — Strong for US remote contracts
5. **CyberSecurityJobsite** — UK/EU focused

### Contract / Freelance
1. **Upwork** — Start with small SOC/cloud security gigs
2. **Toptal** — Requires screening but high pay
3. **Fiverr** — Vulnerability assessments, report writing
4. **Colonial Life / A-LIGN** — Contract SOC roles

### Niche / Communities
1. **r/netsec** — Job postings
2. **r/cybersecurity** — Weekly hiring threads
3. **InfoSec Twitter/X** — Follow hiring managers
4. **Discord — The Cyber Community, TryHackMe, HackTheBox**
