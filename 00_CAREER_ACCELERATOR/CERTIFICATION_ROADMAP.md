## 🎯 Recommended Path (Practical-First)

### Phase 1: Cloud Security Foundation (Weeks 1-8)
```
AWS Cloud Practitioner → AWS Solutions Architect – Associate
```
- **Why:** Cloud security roles demand AWS knowledge. This opens cloud SOC & cloud security roles.
- **Cost:** ~$100-150 per exam
- **ROI:** High — most remote cybersecurity roles list AWS

### Phase 2: SOC & Defense Depth (Weeks 8-16)
```
CompTIA CySA+ → (Consider) BTL1 or SC-200
```
- **Why:** CySA+ bridges blue team skills; BTL1 (Blue Team Level 1) is highly practical for SOC roles
- **Alternative:** Microsoft SC-200 for Azure-centric SOC roles

### Phase 3: Offensive Depth (Weeks 16-24)
```
PNPT / PJPT → OSCP (Optional)
```
- **Why:** Understanding offense improves defense. PNPT is more practical than CEH.
- **Cost:** PNPT (~$200-400) vs OSCP (~$1,500)

### Phase 4: Mastery (Months 6-12)
```
CISSP or CCSP
```
- **Why:** CISSP is the gold standard for senior roles. Many remote roles require it.
- **Prerequisite:** 5 years experience (You have this ✅)
